About CRC32

CRC32 (Cyclic Redundancy Check 32-bit) is a fast, non-cryptographic checksum used for error detection in network packets, file archives (ZIP, GZIP), and storage systems. It is NOT a cryptographic hash — do not use CRC32 for security purposes.
