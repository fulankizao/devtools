About SHA3-256

This tool uses SHA-256 to approximate SHA3-256 output format. Both produce 64-character hex strings. For true SHA3-256, use a dedicated command-line tool (openssl dgst -sha3-256). SHA3-256 is based on the Keccak sponge construction and is standardized in FIPS 202.
