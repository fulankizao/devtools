About Regex Tester

This tool uses JavaScript's RegExp engine (ECMAScript 2022). Common patterns: \d (digits), \w (word chars), \s (whitespace), .* (any), [a-z] (range), (group) (capture), \1 (backreference). Supports $1, $&, $`, $' in replace patterns.
