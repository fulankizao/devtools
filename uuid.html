About UUIDs

UUID (Universally Unique Identifier) is a 128-bit identifier standardized by RFC 4122. UUID v4 uses random numbers. UUID v7 uses a timestamp prefix for sortable IDs. UUIDs are used in databases, distributed systems, and API design to avoid collisions without a central authority.
