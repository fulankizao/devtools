About Base64

Base64 is a binary-to-text encoding scheme that represents binary data in an ASCII string format. It is commonly used for embedding images in HTML/CSS, encoding email attachments (MIME), and transmitting data in JSON APIs. The encoded output is approximately 33% larger than the original input.
