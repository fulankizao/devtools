About SHA1

SHA1 (Secure Hash Algorithm 1) is a cryptographic hash function that produces a 40-character hexadecimal string (160-bit digest). It was originally designed by the NSA and is widely used for data integrity verification and digital signatures. While SHA1 is still found in legacy systems and Git revision identifiers, it is no longer considered collision-resistant for security-critical applications — SHA256 or stronger is recommended for new use cases.
